# Design System Inspired by 52 Launch

## 1. Visual Theme & Atmosphere

52 Launch is launch-velocity — a light corporate system inspired by 52launch.com. The substrate is a cool soft gray (`#f4f5f8`) with pure white cards and a confident deep-indigo primary (`#2b3aa2`) paired with an energetic coral (`#ff5a4e`) — the indigo carries trust and structure; the coral is the rocket-trail signal. Cards have tight `4px` rounding with sharp `1px` borders — the system trades Solamp's friendly 8px corners for a more geometric, engineered feel.

Type is `Inter` for body / UI paired with `Space Grotesk` for display — the same pairing as Monolith but in a light context. Space Grotesk's distinctive open `g` and tight terminals carry display moments at 700 weight; Inter handles body work without compromise. Base size is 15px / 1.55 — slightly tighter than Solamp's 1.6, reinforcing the velocity feel. The page-pattern is the system's most distinctive surface: a `-45deg` repeating diagonal stripe at 18-on / 1-off in the indigo color at 2.5% opacity — reads as launch-trajectory, never crosshatched.

**Key Characteristics:**
- Light substrate (`#f4f5f8`) with pure white cards. Cooler and slightly more saturated than Solamp's gray-white.
- Indigo + coral accent pair (`#2b3aa2` / `#ff5a4e`) — indigo for primary chrome, coral for the energetic / urgent moments (live status, error, accent CTAs).
- Tight `4px` card rounding — geometric, engineered, faster-reading than Solamp's 8px.
- Type pairing: `Inter` body / `Space Grotesk` display. Same shape as Monolith's pairing but in a light context.
- Diagonal stripe background pattern at -45deg in indigo-at-2.5% — the system's signature texture, evoking trajectory.
- Status dots use coral for `.live` with a 1.2s blink — the fastest blink of any light theme, urgency-coded.

## 2. Color Palette & Roles

### Backgrounds
- **Deep** (`#f4f5f8`): Page floor — cool soft gray.
- **Surface** (`#ffffff`): Main content wrapper — pure white.
- **Card** (`#ffffff`): Cards — match surface; differentiate by border.
- **Hover** (`#eaecf1`): Row-hover and de-emphasized surfaces.

### Text
- **Bright** (`#0c0f1d`): Headings, statement text. A deep blue-black — never pure black.
- **Text** (`#2b2f3a`): Body copy. Slate-800 family with a slight blue cast.
- **Muted** (`#6b7280`): Captions, timestamps. Slate-500.

### Accents
- **Indigo** (`#2b3aa2`): The signature primary. Buttons, links, active borders, focus rings. Saturated and confident.
- **Coral** (`#ff5a4e`): The energetic counterpoint. Live status, error state, accent CTA fills, rocket-trail moments.
- **Indigo Glow** (`rgba(43, 58, 162, 0.10)`): Card-hover bloom and focus halo.
- **Coral Glow** (`rgba(255, 90, 78, 0.14)`): The ember-trail wash.

### Borders
- **Border** (`#d9dce4`): Default hairline. Slate-200 with a slight blue cast.
- **Border Active** (`#2b3aa2`): Full indigo on focus / active state.

### Semantic
- **Info** (`#2b3aa2`): Same as indigo accent.
- **Success** (`#10b981`): A confident emerald.
- **Warning** (`#f59e0b`): Soft amber.
- **Error** (`#ff5a4e`): Same as the coral accent — the system collapses error and brand-energy into the same color.

## 3. Typography Rules

### Font Family
- **Primary**: `Inter`, with fallbacks: `system-ui, sans-serif`
- **Display**: `Space Grotesk`, with fallbacks: `Inter, sans-serif`
- **Mono**: `JetBrains Mono`, with fallbacks: `monospace`
- **Terminal**: `JetBrains Mono`, with fallbacks: `monospace`

Same family pairing as Monolith. The difference is context: Monolith uses this pairing in dark brutalism; 52 Launch uses it in light corporate. Space Grotesk's geometric letterforms read as engineering-precision in both contexts.

### Hierarchy
Base 15px / 1.55 — slightly tighter than other light themes, reinforcing velocity.

| Role | Family | Size | Weight | Use |
|---|---|---|---|---|
| Display XL | Space Grotesk | 48–64px | 700 | Page hero |
| Display | Space Grotesk | 32–40px | 700 | Section heads |
| H2 | Space Grotesk | 22–24px | 600 | Subsection heads |
| H3 | Inter | 16px | 600 | Inline subheads |
| Body | Inter | 15px | 400 | Default running text |
| Caption | Inter | 13px | 500 | Card meta, labels |
| Micro | Inter | 11px | 700, uppercase, 0.15em tracking | Section labels, classifications |
| Mono | JetBrains Mono | 13px | 400 | Code, IDs |

### Principles
- **Heavy display weight, tight body leading.** Space Grotesk 700 + Inter at 1.55 — the system reads as fast.
- **Sentence case for display.** All-caps would fight Space Grotesk's distinctive letterforms.
- **Tracking on micro labels at 0.15em.** Slightly tighter than Solamp's 0.12em — labels read engineered.

## 4. Component Stylings

### Cards (`.t-card`)
- Background: pure white.
- Border: `1px solid #d9dce4`, radius `4px` — geometric.
- Hover: border becomes full indigo + `box-shadow: 0 8px 24px rgba(43, 58, 162, 0.08)` — no transform, just elevation.

### Glow (`.t-glow`)
- `box-shadow: 0 0 18px rgba(255, 90, 78, 0.14)` — the ember trail, coral-at-14%. Used on accent/CTA surfaces where the system wants to signal "go."

### Status Dots
- 8×8px circles.
- `.live` — coral fill with coral bloom + **1.2s blink** — the fastest blink in any light theme. Urgency-coded.
- `.idle` — amber.
- `.error` — coral (same as live; context disambiguates).

### Background Pattern (`.t-bg-pattern`)
- `-45deg` repeating diagonal stripe: 18px transparent, 1px indigo-at-2.5%. The system's signature surface — reads as launch-trajectory or speed lines, never as crosshatch.

### Skeleton Shimmer
- 90deg gradient at 1.4s.

### Scrollbar
- 10px width with tight 2px-radius thumbs (vs the more-rounded 5px in Solamp / DHM). The angular thumb continues the geometric vocabulary.

## 5. When To Reach For This Theme

- Startup / launch marketing — the energetic palette and velocity-coded type land directly.
- Founder-facing investor surfaces.
- Product-launch microsites where the visual signal "this is fast / new / shipping" matters.
- B2B SaaS that wants more energy than Solamp's institutional default.

## 6. Avoid This Theme For

- Consumer-luxury or hospitality contexts — the velocity vocabulary fights "savor."
- Audiences that read coral + indigo as fashion-y rather than technical.
- Surfaces requiring extreme calm — 52 Launch is the most energetic of the light themes.
